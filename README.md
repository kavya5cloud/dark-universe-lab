# Dark Universe Lab

A research-learning project focused on computational cosmology,
dark matter reconstruction, weak gravitational lensing,
and diffusion models for scientific machine learning.

## Current Progress

### Project 01
Gaussian Random Fields

Goals:
- Generate synthetic matter density fields
- Understand overdensity maps
- Learn Fourier analysis
- Build intuition for cosmological structure

## Future Milestones

- Weak lensing simulation
- Shear map generation
- CNN-based mass reconstruction
- U-Net reconstruction
- Diffusion prior reconstruction
- Reproduction of CVPR 2026 dark universe mapping paper
